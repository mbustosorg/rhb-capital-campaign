Cached for 60s · Mode: public

Connect a Google Sheet so the gauge reads the raised value live from a spreadsheet cell.

  1. Open your Google Sheet → File → Share → Anyone with the link = Viewer
  2. Copy the Spreadsheet ID from the URL:
    …/spreadsheets/d/SPREADSHEET_ID/edit
  3. Add to your .env file:
SHEETS_MODE=public
SPREADSHEET_ID=your_id_here
RAISED_CELL=Sheet1!B2

Then restart Flask. The cell value will be read every 60 seconds and used as the "raised" amount automatically. Until configured, the manual value below is used.

API Reference
GET/api/fundraiser
Returns current goal and raised amount
curl http://localhost:5000/api/fundraiser
POST/api/fundraiser
Update goal and/or raised. Send any combination of fields.
curl -X POST http://localhost:5000/api/fundraiser \ -H "Content-Type: application/json" \ -d '{"goal": 50000, "raised": 12500}'
POST/api/fundraiser/reset
Reset goal to 10,000 and raised to 0
curl -X POST http://localhost:5000/api/fundraiser/reset
